# EES Specification Index

This index is the single authoritative catalogue of all Eunoia Engineering Specifications. It is updated whenever a document is created, promoted, or deprecated. Never infer a specification's current status from its file content alone — always consult this index.

---

## Status Definitions

| Status | Meaning |
|--------|---------|
| **Draft** | Under active authorship. Not yet approved for implementation. May contain gaps. |
| **In Progress** | Approved and currently being implemented. The specification is binding. |
| **Approved** | Reviewed and signed off by all stakeholders. Ready for implementation to begin. |
| **Implemented** | Fully built, tested, and deployed. Specification is frozen unless a revision cycle opens. |
| **Deprecated** | Superseded by a newer specification or a strategic decision to remove the capability. Retained for audit purposes. |
| **Future** | Scoped and queued but not yet ready for authorship. Placeholder reserved to hold the slot in the numbering sequence. |

## Numbering Convention

Specifications are numbered in order of strategic priority at the time they enter the backlog. Numbers are permanent — a deprecated specification retains its number, and new specifications always receive the next available number. Gaps in the sequence indicate that a specification was removed before being assigned.
